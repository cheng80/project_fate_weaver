# Human Playtest Run 1 Player Sheet v0.1

Status: `[Ready]`  
Date: 2026-07-01

## 테스트 소개

이 테스트는 FateWeaver의 텍스트 모험 한 run이 사람에게 자연스럽게 읽히는지 확인하기 위한 기록지입니다.

완성도 평가가 아니라 "읽으면서 선택할 수 있는가"를 보는 테스트입니다. 정답을 찾으려 하지 말고, 매 turn에서 실제로 고르고 싶은 선택지를 골라 주세요.

## 플레이 방법

- 진행자가 보여 주는 Text MUD run을 turn 순서대로 읽습니다.
- 각 turn에서 선택지 3개 중 하나를 고릅니다.
- 고른 이유를 짧게 적습니다.
- 헷갈리는 순간, 반복처럼 느껴지는 순간, 흥미로운 순간은 바로 표시합니다.
- run이 끝난 뒤 아래 간단 평가를 작성합니다.

## 읽을 때 신경 써야 할 점

- 지금 무엇을 하러 가는지 이해되는가.
- 선택지들이 서로 다르게 느껴지는가.
- 자원 변화가 다음 선택에 영향을 주는가.
- 단서나 징조가 다음 사건으로 이어지는 것처럼 보이는가.
- 같은 사건이 다시 나와도 후속 전개로 느껴지는가, 단순 반복으로 느껴지는가.
- 마지막 Quest Report와 Ending이 내가 한 선택을 정리한다고 느껴지는가.
